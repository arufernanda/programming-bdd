# Programming - BDD (Behavior-driven development)

## Ruby
Installation instructions:
```bash
gem install bundler
bundle install
```
Run linter and tests:
```bash
bundle exec rubocop
bundle exec reek
bundle exec cucumber
```
Debug:

```bash
ruby debug.rb
```
```ruby
binding.pry
```

REPL:
```bash
irb
```
To exit:
```ruby
exit
```
